ALGORITMO
    - conjunto das instruções que realizam uma tarefa

INSTRUÇÃO
    - um comando que visa um objetivo para uma ação simples

CODIGO FONTE
    - é um algoritmo escrito em uma linguagem de programação

LINGUAGEM PROGRAMAÇÃO
    - conjunto estabelecido de instruções conhecidas que serão utilizadas para definir um programa.

PROGRAMA DE COMPUTADOR
    - conjunto de instruções (algoritmo) escrito em linguagem de máquina, que são executados pelo processador

COMPILADOR
    - responsável por entender o código fonte (humano) e escrever um código de máquina
